Run the SQL scripts located in the colorvote_SQL_scripts folder.

A JDBC connection test is located in the package - jdbc.

After running the scripts, simply run the Spring Boot application located in the root package under class name "colorvoteapplication".

Navigate to localhost:8080 and you will be greeted with a simple index page and can click to the link that will take you to the project.

OR, navigate to localhost:8080/showColors
